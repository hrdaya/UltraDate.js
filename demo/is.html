関数概要

UltraDate.prototype.isISOWeekDay(week, day)

  • ISO8601形式でweek週のday曜日かどうかの判定（例：第27週の月曜日）
  • 引数：(Number week, Number day)
  • 戻り値：Boolean

引数

名前 型 説明
week Number
  • 何週目かを指定します
  • 数字にパースできない場合は「0」になります
  • 週番号が前年の週番号となる場合は「0」としてください（1月1日が金曜日、土曜日、日曜日の場合）
  • 週番号が翌年の週番号となる場合は最終週に「1」を足した値にしてください（12月31日が月曜日、火曜日、水曜日）
day Number
  • 曜日を表す 0 (日曜日) ～ 6 (土曜日) の整数を指定します
  • 数字にパースできない場合は「0」になります

戻り値

型 説明
Boolean 現在のUltraDateオブジェクトの日付がISO8601形式でweek週のday曜日なら「true」
