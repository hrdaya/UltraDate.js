diff()

関数概要

UltraDate.prototype.diff(date)

  • 現在のUltraDateオブジェクトの日付から引数「date」までの差分を日、時、分、秒、ミリ秒に変換して返します
  • 例：△△まで残り○○日○○時間○○分○○秒○○○の「○○」の部分をそれぞれ取得します
  • 引数：(UltraDate|Date|String|Number date)
  • 戻り値：Object

引数

名前 型 説明
date UltraDate
Date
String
Number
  • Dateオブジェクトを指定します
  • Stringの場合Dateオブジェクトを作成できる文字列
  • Numberの場合Dateオブジェクトを作成できる数字
  • undefinedの場合は現在の日付

戻り値

型 説明
Object 下記の内容のオブジェクトが返されます
  • after：{Boolean}現在の日付より後かどうか
  • day：{Number}日の差分
  • hour：{Number}時の差分
  • minute：{Number}分の差分
  • second：{Number}秒の差分
  • ms：{Number}ミリ秒の差分

diffMonth()

関数概要

UltraDate.prototype.diffMonth(date)

  • 現在のUltraDateオブジェクトの日付から引数「date」までの差分を月単位で取得します
  • 今月の場合は「0」、翌月の場合は「1」、前月の場合は「-1」
  • 引数：(UltraDate|Date|String|Number date)
  • 戻り値：Number
