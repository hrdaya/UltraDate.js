setRealMonth()

関数概要

UltraDate.prototype.setRealMonth(num)

  • Dateオブジェクトでは月の設定は実際の月から1を引いた値をセットしますが、この関数では実際の月でセットできます
  • チェーンメソッドに対応しています
  • 引数：(Number num)
  • 戻り値：なし(return this;)

引数

名前 型 説明
num Number
  • セットしたい実際の月の数字を指定します
  • 正数の時はミリ秒を増やす
  • 負数の時はミリ秒を減らす

setRoundingTime()

関数概要

UltraDate.prototype.setRoundingTime(time, method)

  • 分単位で丸めた時間にセットします（秒以下は切り捨てて計算）
  • 引数：(time num, method num)
  • 戻り値：なし(return this;)

引数

名前 型 説明
time Number
  • 丸める単位（分）
  • 5,10,15,30,60分の中から選択
  • 上記以外の数字を入力した場合には例外をスローする
method Number
  • 丸め方
  • 0：切り捨て、1：切り上げ、2：近いほうに丸める
  • 数字にパースできない場合や上記数字以外の時は切り捨て
  • 2：近いほうに丸めるときの丸め方
    • 5分のとき：0から2は0、3から4は5、5から7は5、8から9は0
    • 10分のとき：0から4は0、5から9は10
    • 15分のとき：0から7は0、8から14は15
    • 30分のとき：0から14は0、15から29は30
    • 60分のとき：0から29は0、30から59は60
