format()

関数概要

UltraDate.prototype.format(format, eraStrict, locale)

  • 引数「format」で指定されたフォーマットで文字列を返します
  • 引数：(String format, Boolean eraStrict, String locale)
  • 戻り値：String

引数

名前 型 説明
format String
  • 変換する文字列を指定します
  • 「"」ダブルクォーテーションで囲まれた範囲は変換されません
  • 「"」ダブルクォーテーションを文字列として表示するには「""」(連続した「"」)をご使用ください
  • フォーマット文字列については下記を参照してください
eraStrict Boolean
  • 厳格な元号計算「eraStrict」を使用するかどうかを指定します
  • 「true」が指定された時のみ適用されます
locale String
  • 省略された場合はデフォルトのフォーマットが使用されます
  • 指定されたロケールが存在しない場合は「def」が使用されます

戻り値

型 説明
String 変換された文字列

ee 元号計算（フォーマットの設定の「era[year]」）で出された年の右2桁を出力。桁数が足りない場合は左0埋め（00～99）
